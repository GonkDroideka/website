Project Overview

This project demonstrates my ability to deploy cloud infrastructure using modern DevOps practices.

Frontend: HTML5, CSS3 (Responsive Design).

Hosting: Microsoft Azure Static Web Apps (Free Tier).

CI/CD: GitHub Actions (Automated deployment on push).

Security: HTTPS enabled by default via Azure.

How It Works

The source code is managed in this GitHub repository.

When changes are committed to the main branch, a GitHub Action triggers.

The workflow builds and deploys the latest version to Azure automatically.
